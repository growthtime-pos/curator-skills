# Confluence Curation Output Template

## 목적

이 템플릿은 Confluence 문서 큐레이션 결과를 한글로 일관되게 출력하기 위한 기준이다.

출력의 목표는 아래 3가지를 빠르게 판단하게 하는 것이다.

- 어떤 문서가 가장 최신으로 보이는가
- 어떤 문서가 가장 신뢰할 만한가
- 어떤 문서가 중복, 대체, 검토 필요 상태인가

## 기본 출력 원칙

- 최종 출력은 항상 한글로 작성한다
- 점수보다 근거 설명이 더 중요하다
- 최신성과 신뢰도는 분리해서 보여준다
- 애매한 경우 억지로 하나의 정답 문서를 고르지 않는다

## 출력 구조

1. 요약
2. 문서 현황 표
3. 변경 흐름
4. 검토 필요 문서
5. 추천 결론

## 상태 표현

- `최신·신뢰 높음`
- `최신이나 검증 부족`
- `신뢰 높으나 오래됨`
- `중복 가능성`
- `대체됨 가능성`
- `검토 필요`

## 권장 표현

- `~로 보입니다`
- `~가능성이 높습니다`
- `현재 기준으로는`
- `근거가 충분하지 않습니다`
- `추가 확인이 필요합니다`
